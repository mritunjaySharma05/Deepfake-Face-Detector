🛡️ Deepfake Face DetectorA state-of-the-art Deep Learning pipeline for real-time binary classification of facial forgeries. This project was developed as a core portfolio piece for the Associate of Science in AI at Saras AI Institute, focusing on detecting GAN-generated artifacts with high precision.## 📊 Performance at a GlanceFinal Test Accuracy: 94.05%Validation Accuracy: 93.12% (after 5 epochs)Dataset: 140,000 High-resolution Real/Fake images (Kaggle)Inference Latency: &lt; 12ms per image on local hardware## 🧠 Model Architecture &amp; MethodologyThe project employs Transfer Learning using a pre-trained ResNet-50 backbone. By leveraging features learned from ImageNet, the model was specialized for digital forensics.### Custom Classification Head:To adapt ResNet-50 for binary deepfake detection, the following custom head was implemented:Global Average Pooling: Reducing spatial dimensions while retaining critical features.Fully Connected Layer: 2048 nodes → 512 nodes (ReLU activation).Regularization: Dropout (0.3) to prevent overfitting on GAN-specific patterns.Output: Softmax layer for binary classification (REAL vs FAKE).## 💻 Hardware &amp; Engineering SpecsTraining a 140k image dataset requires significant compute. The model was trained on a local workstation to ensure maximum throughput:GPU: NVIDIA GeForce RTX 4060 (8GB VRAM)Storage: Gen 5 SSD (for high-speed data loading)Optimization: PyTorch Mixed Precision Training (torch.cuda.amp)Training Time: ~45 minutes for 5 epochs## 🛠️ Project Structure├── app.py                     # Streamlit web interface for live testing
